# 🚀 NexusERP - Dynamic Enterprise Resource Planning Engine

NexusERP is a next-generation, high-performance Dynamic ERP System built for modern enterprises. It features a fully responsive, utility-first UI dashboard powered by Tailwind CSS and a robust MERN stack architecture to manage corporate operations, track system metrics, and streamline workflow configurations in real-time.

---

## ⚡ Key Features

* **Dynamic Metric Monitoring**: Real-time analytical dashboard with customized widget layouts.
* **Granular Role-Based Access Control (RBAC)**: Secure access management for Admins, Managers, and Employees.
* **Modern UI/UX**: Sleek, fully responsive user interface utilizing Tailwind CSS, built on Vite for blazing-fast performance.
* **Flexible Database Architecture**: Engineered with multi-environment support for instant switching between MongoDB Atlas Cloud and Local MongoDB instances.
* **Secure Authentication Flow**: Token-based secure user sessions with signature verification layers.

---

## 🛠️ Tech Stack

### Frontend
* **Core Framework**: React JS (Vite)
* **Styling**: Tailwind CSS, PostCSS, Autoprefixer
* **State Management**: Redux Toolkit & RTK Query (apiSlice)

### Backend & Database
* **Runtime Environment**: Node.js
* **Framework**: Express.js
* **Database**: MongoDB (Mongoose ODM)
* **Testing API Client**: Thunder Client
